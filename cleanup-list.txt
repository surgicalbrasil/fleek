# Arquivos para remover do repositório

1. debug.js - Duplicado com wallet-monitor.js
2. files\encrypt.js - Utilitário único que já foi usado, não é necessário manter
3. tests\wallet-tests.js - Apenas necessário em ambiente de desenvolvimento

Para remover os arquivos do repositório:

git rm debug.js
git rm files/encrypt.js
git rm tests/wallet-tests.js

E então fazer commit das alterações:

git commit -m 'Remover arquivos desnecessários e duplicados'
