Cadastro no Metaverso

1
Pré-requisitos: Carteira Web3 Metamask. Caso ainda não tenha, siga este tutorial: Tutorial Metamask.
2
Acesse o site: https://gotas.social/.
3
Faça login com sua carteira Web3 Metamask para acessar o painel de resgate.
4
Encontre o código no Certificado de Autenticidade: https://encurtador.com.br/ipw28.
5
Insira o hash da transação no campo destinado para resgatar sua NFT. Atenção: Este código é estritamente pessoal e intransferível. O compartilhamento pode acarretar punições legais.
